# Space Shooter (Python + Pygame)

2D-аркада в жанре вертикального космического шутера с меню, системой прокачки, историей матчей и динамической сложностью.

## Возможности
- Меню игры: старт, история, прокачка, настройки.
- Основной игровой цикл: враги, стрельба, коллизии, очки, монеты.
- Супер-способности с таймером действия.
- Динамическая сложность по мере прогресса.
- Сохранение прогресса в `src/base.json`.

## Требования
- Python 3.10+
- `pygame==2.6.1`
- `pygame-menu==4.5.1`

## Быстрый запуск
### 1) Клонировать/открыть проект
```bash
cd /path/to/ip_proj
```

### 2) Создать виртуальное окружение
#### macOS / Linux
```bash
python3 -m venv venv
source venv/bin/activate
```

#### Windows (PowerShell)
```powershell
py -m venv venv
.\venv\Scripts\Activate.ps1
```

### 3) Установить зависимости
```bash
pip install -r requirements.txt
```

### 4) Запустить игру
```bash
python src/main.py
```

## Управление
- Движение: `↑ ↓ ← →`
- Супер 1: `Z` или `1`
- Супер 2: `X` или `2`
- Пауза: кнопка в левом верхнем углу

## Структура проекта
```text
src/
  main.py              # Точка входа
  game/
    session.py         # Игровая сессия и цикл
    entities.py        # Enemy / Player / Laser
    difficulty.py      # Баланс сложности и шаблоны врагов
    config.py          # Конфиг, клавиши, базовые параметры
  menu.py              # Главное меню
  pause_menu.py        # Меню паузы
  end.py               # Экран победы/поражения
  upgrade.py           # Прокачка
  history.py           # История игр
  settings.py          # Настройки
  methods.py           # Работа с данными и утилиты
  base.json            # Прогресс игрока
```

## Сброс прогресса
Прогресс хранится в `src/base.json`.
Для полного сброса установите:
- `coins = 0`
- `lvl = 1`
- `lvl_of_difficulty = 1`
- `spaceship = "Ships/ship_lvl_1.png"`
- `history = []`
